Privacy Policy - SG Bus Lah!

Privacy Policy / 隐私政策

Last Updated: December 2024

最后更新：2024年12月

English Version

Introduction

SG Bus Lah! (“we”, “our”, or “the App”) is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and safeguard your information when you use our mobile application.

Information We Collect

1. Location Information

  • When Collected: Only when you actively use features that require location (e.g., finding nearby stops, trip tracking)
  • Purpose: To show nearby bus stops/MRT stations and provide real-time trip tracking
  • Storage: Location data is processed locally on your device and is not stored on our servers
  • Background Location: Used only during active trip tracking to provide arrival notifications. You can disable this in Settings.

2. Usage Data

  • What We Collect: App interactions, feature usage patterns, crash reports
  • Purpose: To improve app performance and user experience
  • Third-Party Analytics: We use Google AdMob which may collect anonymous usage data

3. Saved Preferences

  • What We Collect: Favorite stops, saved routes, language preferences, theme settings
  • Storage: Stored locally on your device only
  • Sync: This data is not synced to any cloud service

Third-Party Services

Google AdMob

We use Google AdMob to display advertisements. AdMob may collect:

  • Device identifiers (with your consent via App Tracking Transparency)
  • IP address
  • General location (city-level)
  • App usage data

For more information, see Google’s Privacy Policy

LTA DataMall API

We use Singapore’s Land Transport Authority (LTA) DataMall API to fetch real-time bus arrival and route information. This is public transport data and does not involve personal information.

OneMap API

We use Singapore’s OneMap API for route planning and map services. No personal data is shared with this service.

Data Sharing

We do NOT:

  • Sell your personal data to third parties
  • Share your location history with anyone
  • Store your data on external servers
  • Create user profiles or accounts

Your Rights

You have the right to:

  • Disable Location: Turn off location services for this app in your device Settings
  • Opt-out of Tracking: Deny tracking permission when prompted (iOS 14.5+)
  • Delete Local Data: Uninstall the app to remove all locally stored data
  • Disable Notifications: Turn off notifications in device Settings

Children’s Privacy

This app is not directed to children under 13. We do not knowingly collect personal information from children.

Data Security

We implement appropriate security measures:

  • All API communications use HTTPS encryption
  • No personal data is transmitted to our servers
  • Local data is protected by iOS security features

Changes to This Policy

We may update this Privacy Policy from time to time. We will notify you of any changes by updating the “Last Updated” date.

Contact Us

If you have questions about this Privacy Policy, please contact us at:

  • Email: levi.duan.arts@gmail.com

中文版本

简介

SG Bus Lah!（以下简称"我们"或"本应用"）致力于保护您的隐私。本隐私政策说明了我们在您使用本移动应用时如何收集、使用和保护您的信息。

我们收集的信息

1. 位置信息

  • 收集时机：仅在您主动使用需要位置的功能时（如查找附近站点、行程追踪）
  • 用途：显示附近的公交站/地铁站，提供实时行程追踪
  • 存储：位置数据在您的设备上本地处理，不会存储在我们的服务器上
  • 后台定位：仅在活跃行程追踪期间使用，用于提供到站提醒。您可以在设置中禁用此功能。

2. 使用数据

  • 收集内容：应用交互、功能使用模式、崩溃报告
  • 用途：改善应用性能和用户体验
  • 第三方分析：我们使用 Google AdMob，可能收集匿名使用数据

3. 保存的偏好设置

  • 收集内容：收藏的站点、保存的路线、语言偏好、主题设置
  • 存储：仅存储在您的设备本地
  • 同步：这些数据不会同步到任何云服务

第三方服务

Google AdMob

我们使用 Google AdMob 展示广告。AdMob 可能收集：

  • 设备标识符（需您通过 App Tracking Transparency 授权）
  • IP 地址
  • 大致位置（城市级别）
  • 应用使用数据

更多信息请参阅 Google 隐私政策

LTA DataMall API

我们使用新加坡陆路交通管理局（LTA）DataMall API 获取实时公交到站和路线信息。这是公共交通数据，不涉及个人信息。

OneMap API

我们使用新加坡 OneMap API 进行路线规划和地图服务。不会与此服务共享任何个人数据。

数据共享

我们不会：

  • 将您的个人数据出售给第三方
  • 与任何人分享您的位置历史
  • 将您的数据存储在外部服务器
  • 创建用户档案或账户

您的权利

您有权：

  • 禁用定位：在设备设置中关闭本应用的位置服务
  • 拒绝追踪：在提示时拒绝追踪权限（iOS 14.5+）
  • 删除本地数据：卸载应用以删除所有本地存储的数据
  • 禁用通知：在设备设置中关闭通知

儿童隐私

本应用不面向 13 岁以下儿童。我们不会故意收集儿童的个人信息。

数据安全

我们实施适当的安全措施：

  • 所有 API 通信使用 HTTPS 加密
  • 不向我们的服务器传输个人数据
  • 本地数据受 iOS 安全功能保护

政策变更

我们可能会不时更新本隐私政策。我们将通过更新"最后更新"日期来通知您任何变更。

联系我们

如果您对本隐私政策有任何疑问，请通过以下方式联系我们：

  • 电子邮件：levi.duan.arts@gmail.com
