Set

Set 集合的定义，constructor是构建起，类似init，初始化的传参。this相当于oc中的self.

arr.push(val) 在数组中插入val这个元素
arr.filter(x => x!==val) 过滤掉和val相同的value.
arr.includes(x) 是否包含x这个value
arr.length 返回数组长度.

这是讲解类的构造，Set类的功能是插入的元素，不能有相同的元素。
